快速开始

什么是EZPLM？ ▼
EZPLM是一个专为电子产品研发和制造企业设计的全生命周期管理系统。它整合了项目管理、BOM管理、物料管理、采购管理、库存管理、生产管理等核心功能，帮助企业实现从产品设计到批量生产的全流程数字化管理。
EZPLM适合哪些用户？ ▼
7种用户角色，覆盖企业全流程：

• 系统管理员：管理组织架构、用户权限、系统配置
• 项目经理：项目全生命周期管理、团队协作、进度跟踪
• 工程师：BOM设计、技术文档、工程变更管理
• 采购专员：采购计划、供应商管理、询价比价
• 仓库管理员：库存管理、出入库、质量检验
• 生产人员：生产计划、工单执行、SOP标准作业
• 观察者：只读权限，查看报表和数据分析
如何开始使用EZPLM？ ▼
三步快速上手：

步骤1：注册登录
访问 auth.html 创建账户，系统管理员会为您分配角色和权限

步骤2：完善组织信息
在 organization-settings.html 配置您的组织架构、部门和成员

步骤3：创建第一个项目
在 projects.html 创建项目，开始您的产品管理之旅

典型业务流程

流程1：从项目立项到产品交付 ▼
完整的项目交付流程（8个阶段）：

1. 项目立项 (projects.html)
→ 创建项目 → 分配团队成员 → 设置里程碑

2. 需求与设计 (bom.html)
→ 创建BOM → 添加物料清单 → 设计审查 → BOM审批

3. 采购准备 (procurement.html)
→ 根据BOM生成采购需求 → 发起询价 → 比价选择供应商 → 创建采购单

4. 采购执行 (procurement-tracking.html)
→ 发送采购单 → 跟踪交期 → 收货

5. 质检与入库 (quality-inspection.html, inventory.html)
→ 创建质检单 → 质检检验 → 合格品入库 → 不合格品处理

6. 生产准备 (production.html)
→ 创建生产计划 → 关联BOM和SOP → 物料预留 → 生成工单

7. 生产执行 (work-order-detail.html)
→ 领料出库 → 按SOP生产 → 过程质检 → 成品入库

8. 项目交付 (projects.html)
→ 交付验收 → 成本核算 → 项目归档
流程2：BOM管理和版本控制 ▼
BOM版本管理流程：

1. 创建初始BOM（v1.0）
2. 添加物料明细（规格、数量、位号）
3. 设置替代料（可选）
4. 提交审批
5. 审批通过后BOM生效
6. 如需修改，发起工程变更（ECO）
7. 变更审批后自动生成新版本（v1.1, v2.0）
8. 支持版本对比，查看差异
9. 历史版本可追溯，不可删除
流程3：采购询价和比价 ▼
多供应商询价比价流程：

1. 在采购管理创建询价单
2. 添加需要询价的物料清单
3. 选择目标供应商（可多选）
4. 发送询价单给供应商
5. 供应商在线填写报价（价格、MOQ、交期、库存）
6. 系统自动汇总各供应商报价
7. 比价分析（价格、交期、总成本）
8. 选择最优供应商
9. 一键生成采购单
流程4：质检不合格品处理 ▼
IQC质检流程：

1. 收货后创建质检单
2. 质检员进行检验
3. 录入质检结果

如果合格：
→ 直接入库

如果不合格：
→ 选择处理方式：
  • 退货处理（7-14天）
  • 换货处理（3-7天）
  • 让步接收（影响较小，立即使用）
  • 返工处理（5-10天）
→ 系统自动评估对项目交期的影响
→ 通知相关项目团队
→ 记录供应商质量问题
→ 影响供应商绩效评分
流程5：生产工单执行 ▼
生产执行全流程：

1. 项目经理创建生产计划
2. 选择项目和BOM
3. 设置生产数量和交期
4. 关联SOP标准作业程序
5. 提交审批
6. 审批通过后生成生产工单
7. 系统自动预留物料库存
8. 生产人员领料出库
9. 按照SOP进行生产
10. 实时报工（更新完成数量）
11. 过程质检
12. 完工入库
13. 释放预留库存

常见问题解答

如何设置物料的安全库存？ ▼
在 materials.html 页面，点击物料详情，在编辑页面设置"安全库存"字段。当库存低于安全库存时，系统会自动预警，并生成采购建议。建议根据物料的采购周期和使用频率设置合理的安全库存值。
如何批量导入BOM？ ▼
在 bom.html 页面，点击"导入BOM"按钮，下载Excel模板。按照模板格式填写物料信息（物料编号、名称、规格、数量、位号等），然后上传Excel文件。系统会自动校验数据并导入。如有错误，会给出详细的错误提示。
采购单支持上传哪些附件？ ▼
在新建采购单时，可以上传以下格式的附件：
• Excel表格（.xlsx, .xls）- 适合批量采购物料明细
• PDF文档（.pdf）- 适合费用说明、规格书
• Word文档（.docx）- 适合采购说明
• 图片文件（.jpg, .png）- 适合产品图片
单个文件不超过10MB，支持同时上传多个附件。
库存预留是什么意思？ ▼
库存预留是指为特定订单或项目预先锁定物料数量，防止被其他订单使用。

工作原理：
• 可用数量 = 库存数量 - 预留数量
• 创建生产计划时，系统自动预留物料
• 出库时释放预留，扣减库存
• 取消订单时，释放预留

这样可以确保物料充足，避免超卖。
如何查看项目成本？ ▼
在 project-detail.html 页面，切换到"成本分析"标签页，可以查看：
• BOM物料成本明细
• 采购实际成本
• 生产人工成本
• 外协加工成本
• 总成本汇总
• 成本趋势分析
系统会自动汇总所有相关费用，生成成本报表。
SOP可以导入现有文档吗？ ▼
可以！在 sop-import.html 页面，支持导入以下格式的SOP文档：
• Word文档（.docx）
• PDF文件（.pdf）
• Excel表格（.xlsx）
• Markdown文件（.md）

系统会使用AI自动解析文档内容，提取操作步骤、质量检查点等信息，生成结构化的SOP。您也可以手动调整和完善。
如何设置审批流程？ ▼
系统管理员在 organization-settings.html 中配置审批流程：
1. 选择业务类型（采购单、BOM、生产计划等）
2. 设置审批节点（一级审批、二级审批等）
3. 指定审批人或审批角色
4. 设置审批条件（如金额大于10万需二级审批）
5. 配置通知方式（邮件、站内信）

审批人会在 approval-center.html 收到待审批任务。
不同角色有哪些权限差异？ ▼
7种角色权限说明：

系统管理员：所有功能的完整权限，可以配置系统参数
项目经理：项目、任务、BOM、审批管理权限
工程师：BOM设计、工程变更、技术文档权限
采购专员：采购、供应商、询价比价权限
仓库管理员：库存、出入库、质检权限
生产人员：生产计划、工单、SOP查看权限
观察者：所有模块的只读权限

详细权限矩阵请查看 role-permissions-documentation.html
