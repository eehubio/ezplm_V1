📋 SOP简介
📝 目的与适用范围
本标准作业程序规定了ESP32游戏手柄v2.0的完整组装流程，适用于生产线工人进行产品组装操作。 遵循本SOP可确保产品质量一致性，提高生产效率，减少返工率。
⚠️ 重要提示
• 操作前必须佩戴防静电手环
• 严格按照步骤顺序操作，不可跳步
• 每完成一步后进行自检
• 发现异常立即停止并报告班组长
📝 操作步骤
准备工具和物料
1. 从物料仓库领取所需零部件：主板PCB×1、外壳上盖×1、外壳下盖×1、摇杆模块×2、按键×8、螺丝×6
2. 准备工具：十字螺丝刀、镊子、防静电手套
3. 检查物料外观，确认无破损、变形
4. 将物料按顺序摆放在工作台上
⏱️ 预计耗时：5分钟 ⚠️ 注意：检查物料数量和型号
安装摇杆模块
1. 将左侧摇杆模块对准主板PCB上的J1接口
2. 确保金手指方向正确，轻轻按压至完全插入
3. 听到"咔哒"声表示卡扣锁紧
4. 重复以上步骤安装右侧摇杆到J2接口
⏱️ 预计耗时：3分钟 ⚠️ 注意：避免用力过猛损坏接口
安装按键组件
1. 从左到右依次安装8个按键：L1、L2、SELECT、START、A、B、X、Y
2. 每个按键对准孔位后垂直按下，确保卡扣到位
3. 轻轻按压每个按键测试回弹手感
4. 确认所有按键安装牢固且按压顺畅
⏱️ 预计耗时：8分钟 ⚠️ 注意：按键方向不可装反
组装外壳
1. 将装配好的主板平放在外壳下盖内
2. 确保主板的USB接口、电源开关与外壳孔位对齐
3. 盖上外壳上盖，从四周边缘开始按压确保卡扣扣合
4. 检查外壳接缝处是否平整，无明显缝隙
⏱️ 预计耗时：5分钟 ⚠️ 注意：避免夹到内部线缆
固定螺丝
1. 使用十字螺丝刀拧入6颗M3×8螺丝
2. 螺丝位置：左上、左下、右上、右下、中上、中下
3. 螺丝拧紧力度适中，不可过紧导致外壳变形
4. 检查所有螺丝是否平齐，无突出或松动
⏱️ 预计耗时：6分钟 ⚠️ 注意：螺丝扭矩控制在0.5N·m
功能测试
1. 使用USB线连接手柄到测试电脑
2. 打开测试软件，逐一测试所有按键功能
3. 测试左右摇杆的X/Y轴移动和按压功能
4. 确认蓝牙连接功能正常
5. 所有测试项目通过后，在测试记录表上签字确认
⏱️ 预计耗时：10分钟 ⚠️ 注意：测试必须100%通过
贴标签和包装
1. 在手柄背面贴上产品标签（包含序列号和生产日期）
2. 将手柄放入防静电袋中密封
3. 放入包装盒，加入说明书、USB线、保修卡
4. 封装外箱，贴上物流标签
⏱️ 预计耗时：8分钟 ⚠️ 注意：标签位置准确，无气泡
🔍 质量检查点
✓ 检查点 1：摇杆安装
检查标准：摇杆模块插入到位，无松动；金手指接触良好；摇杆移动顺畅无卡顿。
✓ 检查点 2：按键手感
检查标准：所有按键回弹正常，无粘滞；按压力度一致；按键无晃动。
✓ 检查点 3：外壳组装
检查标准：外壳接缝均匀，缝隙≤0.5mm；表面无划痕、污渍；螺丝拧紧无松动。
✓ 检查点 4：功能测试
检查标准：所有按键、摇杆功能100%正常；蓝牙连接稳定；测试软件显示无错误。
